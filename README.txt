Senior Design: Airplane Testing Apparatus

Introduction: This is a bunch of sensors using the Teensy 3.6 in order to save and store data onto the onboard sd card.

Files:
	bin: Contains the main function used for the program
	test: Contains the functions for each sensor we use
	doc: Contains documentation/findings we've collected while making this.

All codes are written in C++ using the Arduino .ino format.
